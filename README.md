# Quiz-App

## Description:

Quiz Application built with Node js , Express js , Jquery and Sequelize

## Installation:

Node and Sequelize
npm init -y
npm install
server.js
install npm express
npm install sequelize (dont use npm i sequelize)
npm i mysql2
